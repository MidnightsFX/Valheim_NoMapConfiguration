# NoMapTools

This mod aims to fill in some of the existing gaps around no map playthroughs.

It currently provides a few features which are designed to be used with existing other mods to enhance your playthrough.

## Features

- Cartography table functionality requirements
	- Requires X (configurable) number of structures nearby the cartography table for it to function. No more pocketing a cartography table for map viewing/updates
- Vegvisir pings
	- Provides a red ping which lasts for a configurable duration that points to the vegvisir's target location. Especially useful if you find a marker in a dungeon, and want to follow its direction once outside of the dungeon 


## Suggested Mods
- [No Map Printer](https://thunderstore.io/c/valheim/p/shudnal/NomapPrinter/) - Simplifies the map and makes the Map Table the way to update your map
- [Better Cartography Table](https://thunderstore.io/c/valheim/p/nbusseneau/Better_Cartography_Table/) - Allow viewing the map ONLY when using the cartography table
